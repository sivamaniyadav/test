# Login App

A simple Spring Boot web application with a login page.

## Usage

1. Run the application: `mvn spring-boot:run`

2. Open http://localhost:8080 in your browser.

3. Enter user: `siva`, password: `12345` to login.

## Requirements

- Java 17
- Maven

## Troubleshooting

If the server doesn't start, ensure port 8080 is free.
